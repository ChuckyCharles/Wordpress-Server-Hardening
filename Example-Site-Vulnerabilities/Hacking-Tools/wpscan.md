# WPScan Usage

WPScan is a WordPress vulnerability scanner.

## Installation
```
sudo apt install wpscan
```

## Usage
```
wpscan --url https://example.com --enumerate u,vp,vt
```
- `u`: enumerate users
- `vp`: vulnerable plugins
- `vt`: vulnerable themes
